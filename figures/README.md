# Figures

This directory contains publication-ready spectrograms, maps, and derived
figures. Do not add figures containing confidential information, sensitive
coordinates, or third-party material without permission.

For every figure, document:

- a descriptive title and caption;
- the script and data version used to generate it;
- relevant acoustic units, frequency range, and colour scale;
- creator and copyright holder;
- licence and required attribution;
- related publication or dataset, where applicable.

Prefer reproducible figures generated by scripts. Use PNG or SVG for GitHub
display and retain scientific source formats outside the repository if large.
