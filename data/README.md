# Data and metadata

This directory is intended for small, publication-ready CSV tables, metadata,
and data dictionaries. Raw and complete acoustic recordings must not be added
to the Git repository.

## Data catalogue

| Dataset | Description | Period | Area | Access | Licence |
|---|---|---|---|---|---|
| _No datasets published yet_ | | | | On request | |

For each dataset:

1. Create a metadata record from
   [`DATASET_METADATA_TEMPLATE.md`](DATASET_METADATA_TEMPLATE.md).
2. Provide a variable-level dictionary based on
   [`DATA_DICTIONARY_TEMPLATE.csv`](DATA_DICTIONARY_TEMPLATE.csv).
3. State the licence and any restrictions explicitly.
4. Verify that paths, personal data, sensitive coordinates, and restricted
   material have been removed.

## Requesting complete data

Complete recordings are not available through GitHub because of their size.
Researchers may submit a justified request to
[neus.perez@uca.es](mailto:neus.perez@uca.es) or
[servicioacustica.inmar@uca.es](mailto:servicioacustica.inmar@uca.es).
Access is evaluated according to ownership, ethical, legal, and collaboration
conditions and is not guaranteed.
